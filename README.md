# Loop-de-loop

Loop-de-loop is a full loop-hardening prompt for improving other loops with
evidence instead of vibes.

It is built to:

- lock the original loop before changing anything;
- measure challengers against the same task, proof bar, and side-effect limits;
- compare real task value and real run cost, not just prompt length;
- reject fake wins, moved yardsticks, hidden manual rescue, and lucky one-off
  passes;
- preserve a clear promotion or no-promotion decision.

The full loop is in [LOOP.md](LOOP.md).
